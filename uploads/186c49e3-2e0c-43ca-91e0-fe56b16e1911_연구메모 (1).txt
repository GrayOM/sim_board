docker desktop 프로그램 실행
./gradlew clean build 
-> 프로젝트 빌드
docker-compose up --build 
-> docker 컨테이너 실행
docker ps 
-> 컨테이너 확인
docker exec -it board-db-1 mysql -u root -p
-> docker 를 이용해서 mysql 에 접속하겠다는 뜻
-> -it 의 i는 명령어를 입력할수있게끔 하는 옵션
-> -t 는 가상 터미널을 만들어 터미널 환경을 제공해주는 옵션

docker desktop -> docker-compose up -d -> ./gradlew clean build -x test
-> 구동 시킬때 
docker-compose down -> ./gradlew clean build -x test -> docker-compose up
-> 수정 후 해야할 과정
